# GEO-AI 空间选址算法短视频制作包

这个目录是一套面向剪映制作的完整短视频制作包，主题是“GEO-AI 空间选址算法”。

## 已完成内容

- `index.html`：竖屏 9:16 的科技感动态预览页，包含数据网格、HUD、扫描、热力图、竞品盲区、盈利模型、降维打击、关注引导等镜头效果。
- `voiceover.txt`：可直接复制到剪映 AI 配音的中文口播稿。
- `captions.srt`：可直接导入剪映的字幕文件。
- `ai_generation_prompts.md`：12 个镜头的 AI 图片/视频生成提示词。
- `capcut_editing_guide.md`：逐秒剪映时间轴、特效关键词、字幕样式、封面建议。
- `project_manifest.json`：项目结构、导入顺序和风格说明。

## 重要说明

当前执行环境没有剪映/CapCut 桌面端、GUI、命令行导出器，也没有可联网安装视频编码依赖的权限，因此无法在容器里直接打开你的剪映并导出最终 MP4。

为了最大限度完成制作要求，本包提供：

1. 可浏览器预览的 60 秒竖屏动画样片。
2. 可导入剪映的 SRT 字幕。
3. 可粘贴到剪映 AI 配音的口播文案。
4. 可直接照做的剪映逐秒时间轴。
5. 可用于 GPT/AI 视频工具生成素材的镜头提示词。

## 推荐剪映落地流程

1. 按 `ai_generation_prompts.md` 生成 12 段竖屏视频素材。
2. 打开剪映，创建 9:16 项目，导入素材。
3. 导入 `captions.srt`。
4. 复制 `voiceover.txt` 到剪映 AI 配音。
5. 按 `capcut_editing_guide.md` 添加 HUD、扫描、数字雨、闪白、震动、故障、光效转场。
6. 导出 1080×1920、30fps 视频。

## 本地预览

直接用浏览器打开：

```bash
capcut_geo_ai_video/index.html
```

点击“播放浏览器配音”可以使用浏览器自带中文语音同步预览节奏。
